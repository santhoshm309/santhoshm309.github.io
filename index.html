Web Opinion Mining

Natural Language Processing

April 2018 - March 2018
As the title suggests, the aim of the project was to get reviews and label them so as to categorize them. Opinions can be categorized based on the user provided label set and training data. Stanford NLP’s POS tagger and M.A.L.L.E.T Conditional Random Fields were train the model . The project had similar aims to that of an Named Entity Recognizer but with much more advanced label set. The features used to identify the labels were not ellobarate and additional features can yield more accurate results.
